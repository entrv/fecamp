
## 개요
음성이 있는 영상을 한글로 번역 된 srt 파일로 만들어 드립니다.
- 1분당 800원 으로 책정 하고 있습니다.
- 속기 사무소가 아니므로 증거 제출용 으로는 사용이 불가 합니다.

## 사용 기술  
- fast-whisper 의 large-v3 모델을 선택 해서 ai 로 자동 생성 됩니다.
- 그 외는 직접 수정 하셔야 합니다.

## 작업물 결과
- [원어 srt]
- [원어 -> 영어 srt]
- [영어 -> 한글 srt]
- 3 개의 파일을 드립니다. 단, 영어면 원어_srt 가 제공 되지 않습니다.
